# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately by email:

abdeljalil.aitetaleb@gmail.com

Include enough detail to reproduce or understand the issue, such as affected components, versions or commits, impact, reproduction steps, proof-of-concept details, and any relevant logs.

Please do not open a public GitHub issue for suspected vulnerabilities until the issue has been reviewed and a fix or disclosure plan is ready.

## Supported Versions

Security support applies to the latest public release and the default branch unless otherwise stated in release notes.
